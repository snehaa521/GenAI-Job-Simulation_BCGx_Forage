# AI-Powered Financial Chatbot

## 📌 Overview
This project focuses on developing an AI-powered financial chatbot using Python. The chatbot is designed to analyze financial data from 10-K filings and provide structured, user-friendly insights based on predefined queries.

---

## 🏢 Data Source
The data used in this project is taken from the 10-K filings of:
- Microsoft  
- Tesla  
- Apple  

The financial data was manually extracted for analysis.

---

## 📊 Features
- Analyzes financial data from 10-K filings  
- Uses predefined queries to generate insights  
- Provides structured and easy-to-understand outputs  
- Helps simplify complex financial information  

---

## 🛠 Technologies Used
- Python  
- Pandas  
- Excel  

---

## 📈 Example Queries
The chatbot can answer questions such as:
- What is the revenue trend over the years?  
- Which company has higher net income?  
- How does company performance compare?
